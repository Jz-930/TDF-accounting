# TDF Accounting - AI图片生成提词 (Art Assets Prompts)

**使用说明**：
- 提词采用纯英文编写，兼容主流 AI 绘图工具（如 Midjourney, DALL-E 3, Stable Diffusion, Flux 等）。
- 移除了特定软件的指令代码，改为通顺且描写丰富的英文自然语言，能被所有模型正确解析。
- **真实感增强**：增加了大量关于摄影镜头参数（如 85mm, f/2.8）、布光方式（如 侧逆光照、漫反射柔光箱）、材质纹理细节（如 哑光金属、抛光胡桃木、细腻的皮肤纹理与纸张凹痕）和空间景深关系（如 前景极度锐利，背景平滑虚化）的描述，确保生成的图片具有极高的高级感与摄影级别的真实度。
- 为适合网页排版，特别强调了“留白负空间 (negative space)”与具体的画幅比例描述（如宽幅、超宽全景）。
